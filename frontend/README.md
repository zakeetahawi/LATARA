# نظام إدارة علاقات العملاء - واجهة المستخدم

## نظرة عامة
واجهة مستخدم مبنية باستخدام React و TypeScript لنظام إدارة علاقات العملاء.

## المتطلبات
- Node.js (v16+)
- npm or yarn
- Modern web browser

## التثبيت والتشغيل

```bash
# تثبيت المتطلبات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء نسخة الإنتاج
npm run build

# تشغيل الاختبارات
npm test
```
