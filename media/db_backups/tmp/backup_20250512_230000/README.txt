ملف النسخة الاحتياطية لنظام CRM
تاريخ النسخة الاحتياطية: 2025-05-12 23:00:00
=================================

لاستعادة هذه النسخة الاحتياطية:
1. قم بفك ضغط هذا الملف
2. لاستعادة قاعدة البيانات من JSON استخدم:
   python manage.py loaddata db_dump.json
3. أو لاستعادة قاعدة البيانات من SQL استخدم:
   pg_restore --dbname=YOUR_DB_NAME --clean db_dump.sql

4. انسخ محتويات مجلد media إلى مجلد media الخاص بمشروعك
